Windows использует
Обновления Гвардии.
Updates Guard-это системный сервис, работающий в фоновом режиме.
Он опрашивает активность службы обновления Windows.
Если Updates Guard обнаруживает изменение службы Центра Обновления Windows (Wuauserv"), 
это отключит его.
Обновление Guard запускается автоматически.
Это занимает немного системных ресурсов.

Почему Обновления Охранника?
К сожалению, Microsoft использует агрессивный способ установки каждой новой сборки.
Старые уловки не работают.
Обновление охрана всегда поможет остановить обновления.

Вопрос: 
Я не хочу иметь дополнительную услугу в моей системе.
Ответ:  
Откройте меню StopUpdates10, File, Settings. 
Выберите "только для опытных пользователей".
Снимите noх  "Updates Guard is Active".
нажимать OK.

Вопрос: 
Как изменить интервал опроса?
Ответ:  
Создайте текстовый файл с именем "SU10Guard.ini".
Положите это в тот файл:
[Настроить]
CheckProcessSeconds=10

Вопрос: 
Как удалить Updates Guard вручную?
Ответ:  
Запустите "uninstall-guard.bat".

Командная строка:
- отключить обновления Windows:
StopUpdates10.exe /disable

- включить обновления Windows:
StopUpdates10.exe /restore

- перезагрузите компьютер:
StopUpdates10.exe /reboot

- выключение ПК:
StopUpdates10.exe /shutdown

- обновление Защитника Windows:
StopUpdates10.exe /defender
